记忆化

当我们写记忆化的时候要注意几点

  1. 在函数内返回参数而不是再开个数组
  2. sys.setrecursionlimit(1000000)加深
  3. python3.8用不了@cache，用@lru_cache(maxsize=None),记得from functools import lru_cache
from functools import cache
import sys
sys.setrecursionlimit(1000000)
r=int(input())
a=[]
for l in sys.stdin:
    r-=1
    a.append(list(map(int,l.split())))
    if r==0:
        break
@cache
def dfs(i=0,j=0):
    if i==len(a) or j==len(a[i]):
        return 0
    return max(dfs(i+1,j),dfs(i+1,j+1))+a[i][j]
print(dfs())

DP

将dfs视为一棵二叉树，我们为了节省额外空间的开销，可以考虑从二叉树的最底层往上计算，这样就省去了递归中递的步骤
这就是自底向上的递推

import sys
r=int(input())
a=[]
for l in sys.stdin:
    r-=1
    a.append(list(map(int,l.split())))
    if r==0:
        break
# max(dfs(i+1,j),dfs(i+1,j+1))+a[i][j]
for i in range(len(a)-2,-1,-1):
    for j in range(len(a[i])):
        a[i][j]+=max(a[i+1][j],a[i+1][j+1])
print(a[0][0])

例题2 打家劫舍 II

思路

这道题的主要难点是成环，我们可以将将第一家拿出来特判：

  • 如果偷第一家，最后一家也不能偷，可以视为在第二家到倒数第二家之间的无成环打家劫舍

  • 如果不偷第一家，就变成第二家到最后一家的无成环打家劫舍问题
    这样我们就化解了成环难点

题目

你是一个专业的小偷，计划偷窃沿街的房屋，每间房内都藏有一定的现金。这个地方所有的房屋都 围成一圈 ，这意味着第一个房屋和最后一个房屋是紧挨着的。同时，相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警 。
给定一个代表每个房屋存放金额的非负整数数组，计算你 在不触动警报装置的情况下 ，今晚能够偷窃到的最高金额。

提示：

1 <= nums.length <= 100
0 <= nums[i] <= 1000

DFS（会超时）

class Solution:
    def rob(self, nums: List[int]) -> int:
        # 防止一些题目爆栈
        sys.setrecursionlimit(10000000)
        def dfs(i,end,s):
            if i>=end:
                return 0
            if not s:
                return max(dfs(i+1,end,True)+nums[i],dfs(i+1,end,False))
            return dfs(i+1,end,False)
        return max(dfs(1,len(nums)-1,True)+nums[0],dfs(1,len(nums),False))

记搜

将dfs视为一棵二叉树，很明显存在重复计算（重叠子问题），那么就可以使用“备忘录”优化

class Solution:
    def rob(self, nums: List[int]) -> int:
        # 防止一些题目爆栈
        sys.setrecursionlimit(10000000)
        # 在3.9以前的版本没有@cache可使用@lru_cache(maxsize=None)达成一样的效果
        # 当然这里也可以用哈希表手动存
        @cache
        def dfs(i,end,s):
            if i>=end:
                return 0
            if not s:
                return max(dfs(i+1,end,True)+nums[i],dfs(i+1,end,False))
            return dfs(i+1,end,False)
        return max(dfs(1,len(nums)-1,True)+nums[0],dfs(1,len(nums),False))

DP

将dfs视为一棵二叉树，我们为了节省额外空间的开销，可以考虑从二叉树的最底层往上计算，这样就省去了递归中递的步骤
这就是自底向上的递推

一种常见的写法

class Solution:
    def rob(self, nums: List[int]) -> int:
        def robs(i,j):
            dp=nums[i:j]
            # 初始化
            if len(dp)>=2:
                dp[-2]=max(dp[-1],dp[-2])
            for i in range(len(dp)-3,-1,-1):
                dp[i]=max(dp[i+1],dp[i+2]+dp[i])
            return dp[0] if dp else 0
        return max(robs(2,len(nums)-1)+nums[0],robs(1,len(nums)))

实际上回顾DFS的二叉树，我们会发现实际上我们将不存在的底层节点视为0

if i>=end:
    return 0

也就是说我们可以多加入一个0节点即dp.append(0)来代替初始化的特判dp[-2]=max(dp[-1],dp[-2]) if len(dp)>=2
完整代码如下

0节点写法

class Solution:
    def rob(self, nums: List[int]) -> int:
        def robs(i,j):
            dp=nums[i:j]
            dp.append(0)
            for i in range(len(dp)-3,-1,-1):
                dp[i]=max(dp[i+1],dp[i+2]+dp[i])
            return dp[0] if dp else 0
        return max(robs(2,len(nums)-1)+nums[0],robs(1,len(nums)))

DP压缩

我们发现在dp[i]始终只与上两个dp有关，所以只用两个变量压缩

class Solution:
    def rob(self, nums: List[int]) -> int:
        def robs(i,j):
            a,b=0,0
            for i in range(j-1,i-1,-1):
                # python 可以直接写成 b,a=a,max(a,temp+nums[i])
                temp=b
                b=a
                a=max(a,temp+nums[i])
            return a
        return max(robs(2,len(nums)-1)+nums[0],robs(1,len(nums)))

例题三 [SHOI2002] 滑雪

题目描述

Michael 喜欢滑雪。这并不奇怪，因为滑雪的确很刺激。可是为了获得速度，滑的区域必须向下倾斜，而且当你滑到坡底，你不得不再次走上坡或者等待升降机来载你。Michael 想知道在一个区域中最长的滑坡。区域由一个二维数组给出。数组的每个数字代表点的高度。下面是一个例子：

1   2   3   4   5
16  17  18  19  6
15  24  25  20  7
14  23  22  21  8
13  12  11  10  9

一个人可以从某个点滑向上下左右相邻四个点之一，当且仅当高度会减小。在上面的例子中，一条可行的滑坡为 24−17−16−124-17-16-124−17−16−1（从 242424 开始，在 111 结束）。当然 252525－242424－232323－…\ldots…－333－222－111 更长。事实上，这是最长的一条。

输入格式

输入的第一行为表示区域的二维数组的行数 RRR 和列数 CCC。下面是 RRR 行，每行有 CCC 个数，代表高度(两个数字之间用 111 个空格间隔)。

输出格式

输出区域中最长滑坡的长度。

样例 #1

样例输入 #1

5 5
1 2 3 4 5
16 17 18 19 6
15 24 25 20 7
14 23 22 21 8
13 12 11 10 9

样例输出 #1

25

提示

对于 100%100\%100% 的数据，1≤R,C≤1001\leq R,C\leq 1001≤R,C≤100。

from functools import lru_cache
import sys
sys.setrecursionlimit(1000000)
r,c=map(int,input().split())
h=[]
cnt=0
for l in sys.stdin:
    cnt+=1
    h.append(list(map(int,l.split())))
    if cnt==r:
        break
@lru_cache(maxsize=None)
def dfs(x,y):
    ans=0
    for a,b in ((1,0),(-1,0),(0,1),(0,-1)):
        if  0<=x+a<r and 0<=y+b<c and h[x+a][y+b]<h[x][y]:
            ans=max(dfs(x+a,y+b),ans)
    return ans+1
res=0
for i in range(r):
    for j in range(c):
        res=max(dfs(i,j),res)
print(res)
