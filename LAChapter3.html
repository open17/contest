1.4.1. Note on Inner Product Spaces

For ANY vector space 𝑉, there is an abstract definition of inner product ⟨𝐱,𝐲⟩: it must satisfy

  • ⟨𝐱,𝐱⟩≥0, with equality if and only if 𝐱=𝟎
  • ⟨𝐱,𝐲⟩=⟨𝐲,𝐱⟩
  • ⟨𝛼𝐱+𝛽𝐲,𝐳⟩=𝛼⟨𝐱,𝐳⟩+𝛽⟨𝐲,𝐳⟩
